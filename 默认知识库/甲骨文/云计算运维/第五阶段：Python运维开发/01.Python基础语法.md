# 四、Python注释（重点）

## 注释的作用

首先强调一件事：Python代码 => Python解析器 => 机器语言，但是注释经过了Python的解释器并不会解析与执行。因为其主要就是进行代码的注释。

注释作用：<font style="color:rgb(216,57,49);">提高代码的阅读性</font>

![1763180474267-e1d7133d-e527-4d1a-99e9-2126b8379adb.png](./img/T77Y3RHvky_L-VlG/1763180474267-e1d7133d-e527-4d1a-99e9-2126b8379adb-231047.png)

在我们编写Python程序时，为了提高程序的可读性，强烈建议大家为<font style="color:rgb(216,57,49);">核心代码</font>添加注释信息。

## Python注释的基本语法

### 单行注释

单行注释，以"#"(Shift + 3)号开头，只能注释一行内容

```python
# 注释内容
```

示例代码：

第一种：代码行的上面（往往用于标注程序功能）

```python
# 输出Hello World字符串
print('Hello World')
```

第二种：放在代码的后面(代码后面保留2个空格) => （往往用于标注程序执行结果）

```python
print('Hello World')  # 输出Hello World字符串
```

### 多行注释

多行注释：可以同时注释多行代码或程序，常用于代码块的注释，编写程序设计思路

基本语法：

```python
"""
注释内容
第一行
第二行
第三行
"""
```

或

```python
'''
注释内容
第一行
第二行
第三行
'''
```

示例代码：

```python
"""
Hi, 大家好
从今天开始，我们将一起学习Python这门语言
"""

'''
Hi, 大家好
从今天开始，我们将一起学习Python这门语言
'''
print('Hi, 大家好')
print('从今天开始，我们将一起学习Python这门语言')
```

### PyCharm注释小技巧（快捷键）

在PyCharm中，我们可以使用`Ctrl + /斜杠`来对代码或程序进行快速注释。

小结：

注释的作用？答：<font style="color:rgb(216,57,49);">提高代码的可阅读性</font>

注释一共分为两种？答：<font style="color:rgb(216,57,49);">单行注释</font> 和 <font style="color:rgb(216,57,49);">多行注释</font>

快捷键 => (<font style="color:rgb(216,57,49);">Ctrl + /</font>)

## 代码提示

在PyCharm中，当我们输入Python关键字中的前2~3个字符，其会自动进行代码提示。这个时候，我们只需要按回车即可以快速的输入某个内容。

## 代码保存

编写代码时，一定要养成一个好的习惯，使用`Ctrl + S`快速对代码进行保存操作。

如果不小心删除了某行代码，这个时候我们可以快速按`Ctrl + Z`就可以快速进行恢复。每按一次就撤销一次，如果撤销多了，怎么办？

## 代码格式化处理

在写代码过程中，如果代码不够规范，没有关系，可以使用 Ctrl + Alt + L，实现快速格式化！

## if...elif...else多条件判断结构（多分支结构）

...

```python
    ...
else:
    如果以上所有条件判断都不满足时，则执行这个语句段
```

## range方法（函数）

for循环主要场景都是<font style="color:rgb(216,57,49);">实现容器类型遍历</font>，for循环<font style="color:rgb(216,57,49);">其实也可以结合range()函数，实现循环指定次数（如循环5次、循环10次）</font>。

Python2 range() 函数返回的是列表，而在Python3中 range() 函数返回的是一个<font style="color:rgb(216,57,49);">可迭代对象</font>（类型是对象，<font style="color:rgb(216,57,49);">只能通过</font><font style="color:rgb(216,57,49);">for循环</font><font style="color:rgb(216,57,49);">遍历输出</font>），而不是列表类型， 所以打印的时候不会打印列表。（由于我们还未学习面向对象，为了方便大家理解，你可以简单的将其理解为一个序列结构）

主要作用：用于生成一段连续的内容，从0到9

基本语法：

```python
range(stop)
range(start,stop)
range(start, stop[, step])

start: 计数从 start 开始。默认是从 0 开始。例如range（5）等价于range（0， 5）;
stop: 计数到 stop 结束，但不包括 stop。例如：range（0，5） 是 [0, 1, 2, 3, 4] 没有 5
step：步长，默认为1。例如：range（0，5） 等价于 range(0, 5, 1)

range(1,11,2) ：获取1-10之间所有奇数，1、3、5、7、9
```

> range有一个口诀：<font style="color:rgb(216,57,49);">只顾头来尾不管</font>，包含头部信息，但是不包含尾部信息，如range(10)，则返回0~9之间的序列，又比如range(0, 5)代表返回0 ~ 4之间的序列。

案例：for循环与range方法，使用for循环，循环5次

```python

```

## 扩展：for循环嵌套结构（了解）

简单来说就<font style="color:rgb(216,57,49);">是一个</font><font style="color:rgb(216,57,49);">for循环</font><font style="color:rgb(216,57,49);">中，又出现了另外一个for循环结构 </font>=> for循环嵌套

for循环嵌套应用场景：① 实现二维图形的打印 ② 九九乘法表打印 ③ 可以处理<font style="color:rgb(216,57,49);">较为复杂的循环结构</font>

基本语法：

```python

    for j in range(start, stop):
        要循环的内容
```

for循环嵌套执行流程与工作原理：① for循环嵌套到底需要循环多少次 ② for循环嵌套如何执行的？如果外层循环循环1次，内层循环要循环多少次。

```python
'''
定义一个循环，外层循环2次，内层循环3次。问总循环次数？
结论1：总循环次数 = 外层循环次数 * 内层循环次数
结论2：执行过程中，外层先执行1次，内层执行N次（类似时钟，分钟移动1格，秒针循环1圈）
'''
# 1. 定义外层循环结构
for i in range(1, 3):
    # 2. 定义内层循环结构
    for j in range(1, 4):
        print('*')
```

```python
print(str1.find('and'))
```

案例：使用input方法输入任意一个文件名称，求点号的索引下标

```python
'''
find()：默认从左向右提取，提取关键词首次出现的位置
rfind()：默认从左向右提取，提取关键词最后一次出现的位置
'''
# 第一步：获取文件名称
filename = input("请输入您要上传文件的名称：")
# 第二步：通过rfind获取最后一个点号位置
pos = filename.rfind('.')  # 1.png => 1
# 第三步：提取文件名称以及文件后缀（切片）
name = filename[:pos]  # filename[:1] => 1
postfix = filename[pos:]  # filename[1:] => .png
print(f"文件名称：{name}")
print(f"文件后缀：{postfix}")
```

### 字符串的替换方法

所谓<font style="color:rgb(216,57,49);">修改字符串</font>，指的就是通过函数（方法）的形式<font style="color:rgb(216,57,49);">修改字符串中的数据</font>。

☆ replace()方法

基本语法：

```python
字符串.replace(要替换的内容, 替换后的内容, 替换的次数-可以省略)

替换次数，代表替换多少次，可以省略，省略后就代表替换所有！！！
```

案例：编写一个字符串，然后把字符串中的linux替换为python

```python
str1 = 'hello linux and hello linux'
# 把字符串中所有linux字符替换为python
print(str1.replace('linux', 'python'))
# 把字符串中的第一个linux进行替换为python
print(str1.replace('linux', 'python', 1))
# 把and字符串替换为&&
print(str1.replace('and', '&&'))
```

目前在工作中，replace主要用于实现关键字替换或过滤功能。北京 ==> BJ，论坛关键字过滤

☆ split()方法

作用：<font style="color:rgb(216,57,49);">对字符串进行切割操作，返回一个list列表类型的数据</font>

```python
str1 = 'apple-banana-orange'
print(str1.split('-'))
```

☆ join()方法

作用：<font style="color:rgb(216,57,49);">和split()方法正好相反，其主要功能是把序列拼接为字符串</font>

```python
字符串.join(数据序列)
```

案例：把水果列表\['apple', 'banana', 'orange']拼接成'apple-banana-orange'

```python
list1 = ['apple', 'banana', 'orange']
print('-'.join(list1))
```
