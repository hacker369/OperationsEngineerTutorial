# 05.电商项目部署

# 一、NiuShop 商城

## NiuShop 电商项目概述

NiuShop 是基于 PHP 语言开发的一款开源的电子商城系统，这个系统的运行需要 LAMP/LNMP 支持！

Niushop 系统是应用 ThinkPHP 框架开发的完善的电商系统，拥有完善的商品机制，包含普通商品，虚拟商品，同时完善了虚拟商品的核销机制，营销活动支持拼团，分销，多门店砍价等新时代的商业模式，配送方式支持物流配送，门店自提，本地配送，整合了当前电商系统的各种营销以及商业方案，同时支持插件开发，系统完善，成为新时代开源商城的首选。

功能特点：

商品功能：支持普通商品，虚拟商品

核销功能：支持自提核销，虚拟二维码核销功能

营销活动：支持拼团，积分兑换，分销功能

配送方式：物流配送，门店自提，本地配送

多门店：支持门店商品，收银台功能

页面装修：支持平台与店铺手机端自定义页面设计

uniapp 开发：采用一端开发多端使用的 uniapp 开发机制可以灵活编译

## NiuShop 集成开发环境说明

NiuShop 版本要求：

```shell
Nginx/Apache
PHP 7.4
MySQL 5.6~8.0
Redis 支持
```

LNMP 为例：两种方式

① 纯手工部署 => Nginx/MySQL/PHP/更改配置

② 宝塔面板 => 提供一套集成开发环境，LNMP/Java/Go 环境

## 宝塔面板管理命令（了解）

我们可以通过在 Linux 中输入相关的命令去管理宝塔面板。

启动宝塔：

```shell
# bt start
```

停止宝塔：

```shell
# bt stop
```

重启宝塔：

```shell
# bt restart
```

检查宝塔状态：

```shell
# bt status
```

升级宝塔：

```shell
# bt update
或者
# bt update <版本号>
```

卸载宝塔：

```shell
# bt stop && rm -rf /www/server/panel && rm -f /etc/init.d/bt
```

宝塔管理：

```shell
# bt

输入bt后，会显示一个交互式菜单，可以用于执行常见操作，如：
查看面板状态
修改面板端口
重置面板密码
```

***

**服务管理：可以使用宝塔命令管理我们安装的服务，也可以使用 systemctl 命令去管理服务！但是不要混合着用！！！（前提是得安装了下面的服务）**

Nginx：

```shell
启动Nginx：
# /etc/init.d/nginx start

停止Nginx：
# /etc/init.d/nginx stop

重启Nginx：
# /etc/init.d/nginx restart

测试Nginx配置文件是否正确：
# /etc/init.d/nginx -t
```

MySQL：

```shell
启动MySQL
# /etc/init.d/mysqld start

停止MySQL
# /etc/init.d/mysqld stop

重启MySQL
# /etc/init.d/mysqld restart
```

PHP

```shell
重启PHP服务（以php 7.4 为例）
# /etc/init.d/php-fpm-7.4 restart

检查php-fpm状态
# /etc/init.d/php-fpm-7.4 status
```

数据管理

```shell
如果忘记面板的用户名或密码，可以运行以下命令查看：
# bt default

修改面板的访问端口（如改为8888）
# bt 8

重置面板管理员密码，如果忘记了面板密码，可以通过以下命令重置：
# bt 5
按提示输入新密码
```

日志查看

```shell
查看网站访问日志：（以站点example.com为例，记录在Nginx上）：
# cat /www/wwwlogs/example.com.log

MySQL错误日志：
# cat /www/server/data/mysql.err

PHP错误日志：
PHP错误日志路径（以PHP7.4为例）：
# cat /www/server/php/74/var/log/php-fpm.log
```

项目目录

```shell
网站文件目录：
# cd /www/wwwroot/

数据文件目录：
# cd /www/server/data/
```

配置防火墙开放面板端口

```shell
如果无法访问宝塔面板，可能是防火墙未开放对应端口，以下命令可以开放宝塔默认端口（8888）
CentOS 防火墙：
# firewall-cmd --zone=public --add-port=8888/tcp --permanent 
# firewall-cmd --reload

Ubuntu 防火墙
# ufw allow 8888
# ufw reload
```

**<font style="background-color:#FBDE28;">搭建环境时可以考虑暂时关闭防火墙（因为宝塔面板启动后，默认会开启防火墙！！！）</font>**

```shell
# systemctl stop firewalld
# systemctl disable firewalld
```

原因：因为 NiuShop 电商平台设置了一个伪静态技术（rewrite）

NiuShop 官方参考文档：<font style="color:#117CEE;">https://www.kancloud.cn/niucloud/niushop\_b2c\_v5/3037621</font>

解决办法：

![1764558872649-da38ff2e-6128-4ace-81d6-598c9e22de28.png](./img/wkhCIeYXYDRrzAF5/1764558872649-da38ff2e-6128-4ace-81d6-598c9e22de28-482619.png)

![1764559068777-cf348f58-ea4f-4941-86ac-5fddf7fadd0e.png](./img/wkhCIeYXYDRrzAF5/1764559068777-cf348f58-ea4f-4941-86ac-5fddf7fadd0e-032274.png)

```shell
location / {
    if (!-e $request_filename) {
        rewrite ^(.*)$ /index.php/$1 last;
        break;
    }
}

解析：location / 代表任意请求，if判断，-e（exists缩写）判断请求页面（$request_filename）是否存在	/shop/index/index.html相当于$request_filename
rewrite 重写模式 => rewrite 旧链接 新链接 last/break;
/index.php/shop/index/index.html

location /h5 {
   if (!-e $request_filename) {
       rewrite ^/(.*) /h5/index.html last;
       break;
   }
}

解析：location /h5代表只要用户访问链接中包含/h5，则匹配到location规则
如果访问的手机端页面不存在，则通过rewrite重写到/h5页面
```
