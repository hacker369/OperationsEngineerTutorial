# 学习总结

1. 简单介绍一下Java语言

Java语言是一门计算机高级语言，是面向对象的语言。有封装、继承、多态的特点。

2. 什么是JDK？

JDK就是Java开发工具包，是提供给程序员使用的。如果电脑上安装了JDK，就不需要安装JRE了。包括：JRE和一些工具。

3. 什么是JRE？

JRE就是Java运行时环境，它提供了Java程序运行时需要的类库。包括：JVM和一些核心类库。

4. 什么是JVM？

JVM就是Java虚拟机，用来解释执行Java代码。

5. JDK、JRE和JVM三者的关系？

JDK = JRE + 工具集

JRE = JVM + 核心类库

6. 如何配置环境变量？
   1. 在高级系统设置环境变量中，创建 JAVA\_HOME，值是JDK的安装路径
   2. 编辑 path，在里面添加 %JAVA\_HOME%/bin
7. 如何检测环境环境是否配置正确？
   1. 打开cmd窗口
   2. 输入java、javac能正常运行就可以
8. 解释一下main方法？

main方法是程序的入口，它的写法固定。

9. 解释一下什么是变量？

变量就是存储数据的容器，它里面存放的内容是可以改变的。

10. 定义变量的语法格式？

`数据类型 变量名 = 变量值;`

11. Java中数据类型分为？

Java中数据类型分为：基本数据类型和引用数据类型。

基本数据类型：byte、short、int、long、float、double、boolean、char

引用数据类型：类、接口、数组

12. String类型属于基本类型？还是引用类型？

String属于引用类型，因为String是一个类。

13. boolean类型的值有哪些？

boolean类型的值只有 true、false。

14. 比较两个数是否相等使用什么符号？

比较两个数是否相等使用 ==

15. 逻辑运算符中，与 和 或 的规律是什么？

与：两边都是真，结果才是真；如果有一个是假，结果就是假

或：两边只要有一个是真，结果就是真；两边都是假，结果才是假

16. Java中的循环有哪些？

Java中的循环有：for循环、while循环、do-while循环

17. 循环包含哪些部分？

初始条件、循环条件、循环体、迭代部分

18. 说一下你对数组的了解？

数组就是存储很多同类型的数据；数组中的元素是有序的；数组的长度一旦给定就不能改变了；数组中的元素通过索引获取，索引是从0开始的。

19. 描述面向过程和面向对象的核心思想？

面向过程，强调的是功能行为，以函数为最小单位，考虑怎么做；

面相对象，强调具备功能的对象，以类/对象为最小单位，考虑谁来做。

20. 定义类的关键字是？类中包含哪些内容？

定义类的关键字是 class。类中可以包含：属性、方法。

21. 定义方法的语法格式？

```java
权限修饰符 返回值类型 方法名(形参列表){
    方法体
}
```

22. 权限修饰符有哪些？

权限修饰符有：public、protected、缺省、private，我们用的最多的是public、private

23. 对于实体类我们如何去封装？
    1. 将属性私有，也就是用private修饰属性
    2. 提供属性的set、get方法
24. 在项目中，我们编写一个实体类会写哪些内容呢？
    1. 写私有的属性
    2. 生成有参、无参的构造方法
    3. 生成属性的set、get方法
    4. 生成toString方法
25. 构造器的作用？
    1. 创建对象
    2. 给对象的属性赋值
26. 继承的关键字是？继承有什么好处？
    1. 继承的关键字是 extends
    2. 继承可以提高代码的复用性、可扩展性
27. 描述一下方法的重写

方法的重写指的是：子类对于从父类中继承过来的方法如果不满意，可以对方法进行重新编写。

28. 多态的写法格式

`父类类型 对象名 = new 子类();`

29. 定义接口的关键字是？接口的作用？
    1. 定义接口的关键字是 interface
    2. 接口就是用来定义一些规范，然后由一些实现类去实现接口
30. List集合中存放数据的特点？

List集合中存放的数据是有序、可重复的，通过索引可以获取集合中的数据。

31. List集合常用的实现类及常用的写法？

List集合常用的实现类是 ArrayList，写法：`List<类型> list = new ArrayList<>();`

32. Scanner的作用是什么？常用的方法有？
    1. Scanner可以用来读取用户在控制台输入的数据
    2. 用next()读取字符串
    3. 用nextInt()读取整数
    4. 用nextDouble()读取小数
33. 创建数据库的sql语句？

`create database 数据库名字 default character set utf8;`

34. 创建表的sql语句？

```sql
create table 表名(
  字段1 数据类型 [约束],
  字段2 数据类型 [约束],
  ...,
  字段n 数据类型 [约束]
);
```

35. 添加数据的sql语句？

`insert into 表名 values (值1, 值2, ..., 值n);`

36. 修改数据的sql语句？

`update 表名 set 字段1=值1, 字段2=值2, ..., 字段n=值n where 条件;`

37. 删除数据的sql语句？

`delete from 表名 where 条件;`

38. 查询所有数据的sql语句？

`select * from 表名;`

39. 查询id=3的数据？

`select * from 表名 where id = 3;`

40. 查询id是1或者2或者3的数据？

`select * from 表名 where id in (1,2,3);`

41. 查询年龄在20-30之间的数据？

`select * from 表名 where age between 20 and 30;`

42. sql中怎样去除重复数据？

使用关键字 distinct。

43. sql中排序的关键字？

使用 order by 字段名 asc|desc，asc 表示升序，desc 表示降序

44. 查询表中名字包含"伟"的数据？

`select * from 表名 where name like '%伟%';`

45. 怎样分组查询？

使用关键字 group by 字段。

46. 分页查询数据，每页显示10条，查询第6页的数据，应该怎么写sql？

`select * from 表名 limit 50, 10;` 公式：(页数-1)\*每页条数

47. 聚合函数有哪些？
    1. count：统计个数
    2. sum：求和
    3. avg：求平均数
    4. max：求最大值
    5. min：求最小值
48. 关联查询有哪几种，分别代表什么意思？
    1. 内连接：两张表中能关联上的数据才能查询出来
    2. 左连接：以左表为主，左表的数据都会查询出来，右表中关联不上的数据显示为null
    3. 右连接：以右表为主，右表的数据都会查询出来，左表中关联不上的数据显示为null
49. JDBC是做什么的？

JDBC 就是Java数据库连接，我们可以通过JDBC技术去操作数据库。

50. 通过JDBC实现查询的步骤是？
    1. 加载数据库驱动
    2. 获取数据库连接
    3. 编写sql语句
    4. 创建预编译语句对象
    5. 给sql中的占位符赋值，并执行sql，得到结果集
    6. 遍历结果集
    7. 关闭连接
51. 介绍一下DBUtils是什么？

DBUtils就是对JDBC做了一些封装，能够让我们操作数据库变得更简单。

52. 使用DBUtils查询数据库的步骤？
    1. 获取数据库连接
    2. 编写sql语句
    3. 创建QueryRunner对象
    4. 调用方法执行sql语句
    5. 关闭连接
53. 在DBUtils中，BeanListHandler 和 BeanHandler 分别是什么含义？
    1. BeanListHandler：表示将查询到的一条条的数据封装为一个个的对象，然后将这些对象放入 List 集合
    2. BeanHandler：表示将查询到的一条数据封装为一个对象

60. 说出JS中的4个事件
    1. onload事件：页面加载完事件
    2. onclick事件：鼠标单击事件
    3. onchange事件：内容改变事件
    4. onblur事件：鼠标失焦事件
    5. onfocus事件：鼠标聚焦事件
61. 编写Servlet的步骤是？
    1. 编写一个类继承HttpServlet
    2. 重写service方法
    3. 通过注解@WebServlet配置访问路径
62. 怎样获取请求参数？怎样往request域对象中放数据？
    1. 通过`request.getParameter("参数名");`去获取参数
    2. 通过`request.setAttribute(键, 值);`给request中存储数据
63. 请求转发的特点？
    1. 请求转发是一次请求
    2. 请求转发地址栏的地址不会发生改变
    3. 请求转发可以共享request域对象中的数据
    4. 可以转发到WEB-INF目录
    5. 不可以转发到项目以外的资源
64. 请求重定向的特点？
    1. 请求重定向是两次请求
    2. 浏览器地址栏地址会发生改变
    3. 不能共享request域对象中的数据
    4. 不能重定向到WEB-INF目录
    5. 可以重定向到项目以外的资源
65. JSTL标签库的使用步骤？
    1. 项目中导入JSTL的两个jar包
    2. 在要使用JSTL标签库的页面中引入相应的标签库
